# Front-end-Enginering

Bem-vindo ao nosso projeto de melhoria de acessibilidade e experiência do usuário para o site do [IMREA](https://www.imrea.org.br/) em parceria com o **Hospital das Clínicas de São Paulo**. Desenvolvido como proposta acadêmica com foco em **design centrado no ser humano**, tecnologia assistiva e usabilidade.

---

## 🚀 Objetivo do Projeto

O Reabilita+ visa modernizar e facilitar o acesso aos serviços de reabilitação oferecidos pelo IMREA, garantindo:

- 🧑‍🦽 Acessibilidade digital real e inclusiva
- 📱 Interface adaptada para diferentes dispositivos
- 🗺️ Localização facilitada de unidades
- 🔐 Acesso simplificado para pacientes e familiares
- ❤️ Navegação amigável, humana e acolhedora

---

## 🧠 Tecnologias & Ferramentas Utilizadas

| Ferramenta | Descrição |
|-----------|-----------|
| 🎨 **Figma** | Protótipos de alta fidelidade, responsivos |
| 🧩 **Design System** | Cores institucionais (HC + IMREA), ícones acessíveis |
| 🧠 **UX Research** | Pesquisa com base em boas práticas de acessibilidade |
| 🧑‍⚕️ **Conteúdo especializado** | Inspirado nas rotinas e necessidades reais de pacientes |

---

## 🎯 Funcionalidades em Destaque

✅ Mapa interativo para localizar unidades  
✅ Login seguro com CPF e senha  
✅ Estrutura pronta para inclusão de leitores de tela  
✅ Navegação com teclado (acessibilidade AA)  
✅ Paleta de cores com bom contraste (W3C compliant)  
✅ Favicon personalizado com identidade visual moderna
